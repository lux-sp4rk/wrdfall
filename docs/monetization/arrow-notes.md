# Wordfall — Arrow Notes (ethical monetization)

## Pricing ("Clarity" model)
- Base game: **$4.99** one-time
- Theme packs: **$1.99** each (50–100 puzzles)
- Optional: **"Master Weaver" pass**: **$14.99** (all current + future packs)
Rationale: seniors often prefer "buy it and own it"; subscriptions feel like a leak.

## Giftable bundles
1) **The Hearth & Home** — Kitchen Traditions + Grandma’s Garden
2) **The Grand Tour** — European Capitals + Great Landmarks + Vintage Postcards
3) **The Naturalist’s Library** — Woodland Birds + Trees of the World + Wildflowers

## App Store listing angles
Description bullets:
- Pure Focus: no ads, no banners
- Designed for Sight: large, high-contrast text
- Hand-crafted puzzles (avoid gibberish)
- Offline support
- Honest value: no gems/energy timers

Keywords:
- Senior word game
- Cognitive health
- Ad-free puzzles
- Accessible gaming
- Relaxing vocabulary

## Outreach pitch (activity directors)
Offer a few free codes for centers to test. Position it as quiet, ad-free, clarity-first, good for group "community solve" or individual calm time.

## Ethical stance
Avoid variable-ratio reinforcement; sell static value (content for a flat fee).
